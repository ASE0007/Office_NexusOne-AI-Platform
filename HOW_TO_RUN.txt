Your project is at: D:\PROJECT\nexusone

═══════════════════════════════════════════════════════════
STEP 1 — Add PostgreSQL to PATH (run once)
═══════════════════════════════════════════════════════════
Open VS Code terminal and run:

  setx PATH "%PATH%;C:\Program Files\PostgreSQL\18\bin"

Then CLOSE and REOPEN the terminal.

═══════════════════════════════════════════════════════════
STEP 2 — Create Database (run once)
═══════════════════════════════════════════════════════════
  "C:\Program Files\PostgreSQL\18\bin\psql.exe" -U postgres

Inside psql type:
  CREATE DATABASE nexusone_db;
  CREATE USER nexusone_user WITH PASSWORD '1234';
  GRANT ALL PRIVILEGES ON DATABASE nexusone_db TO nexusone_user;
  \q

═══════════════════════════════════════════════════════════
STEP 3 — Backend Setup (run once)
═══════════════════════════════════════════════════════════
  cd D:\PROJECT\nexusone\backend
  rmdir /s /q venv
  python -m venv venv
  venv\Scripts\activate
  pip install -r requirements\base.txt
  python manage.py migrate
  python manage.py create_demo_data

═══════════════════════════════════════════════════════════
STEP 3B — Run Automated Tests (recommended, takes ~30 seconds)
═══════════════════════════════════════════════════════════
  python manage.py test tests

  You should see "OK" and "Ran 59 tests". This confirms every module
  (CRM, Projects, Tasks, Billing, Support, HRM, Inventory, Calendar, AI,
  and data isolation between companies) actually works before you start
  building on top of it.

═══════════════════════════════════════════════════════════
STEP 4 — Start Backend (every time)
═══════════════════════════════════════════════════════════
  cd D:\PROJECT\nexusone\backend
  venv\Scripts\activate
  python manage.py runserver

  → Backend runs at: http://127.0.0.1:8000

═══════════════════════════════════════════════════════════
STEP 5 — Start Frontend (every time, new terminal)
═══════════════════════════════════════════════════════════
  cd D:\PROJECT\nexusone\frontend
  npm install
  npm run dev

  → Frontend runs at: http://localhost:3000

  Optional — run frontend tests:
  npm test

TO ENABLE AI FEATURES (Optional)
═══════════════════════════════════════════════════════════
Open: D:\PROJECT\nexusone\backend\.env
Change: OPENAI_API_KEY=sk-your-openai-api-key-here
To:    OPENAI_API_KEY=sk-YOUR-REAL-KEY-HERE

Then restart backend. AI works without key but with limited responses.

═══════════════════════════════════════════════════════════
TO ENABLE ERROR TRACKING (Optional)
═══════════════════════════════════════════════════════════
Backend: add SENTRY_DSN=your-dsn-here to backend\.env
Frontend: add NEXT_PUBLIC_SENTRY_DSN=your-dsn-here to frontend\.env.local
Both are no-ops (app works exactly as before) if left blank.
